About


Options are financial instruments derived fromvarious underlying assets, assuring flexibility andagility to portfolio managers. Options became an indispensable asset in modern investmentmanagement. Nevertheless, the complicated nature of these instruments results in more uncertaintyin setting thecorrect price of the options. The recent advances in Machine Learning (ML) promisesmore efficient and accurate modeling of financial processes, compared to the heuristics basedhistorical financial data analysis. Therefore it is a natural choice to apply ML techniques to improvethe quality of option pricing. The task of the student is to propose a machine learning approachto improve the quality of decision on option pricing. The detailed task list of the student is as follows:

  • Conduct a literature survey of the option pricing models, including the BlackScholes and the binomial pricing models.
  • Review the MLmethods, including the Kernel Methods, Regression Techniques and Deep Learning. Provide an overview of option pricing approaches using machine learning methods.
  • Design a testing framework that is suitable to evaluate the accuracy and performance of ML methods for option pricing. Propose a proper data set to be used during these tests, implement the data acquisition process in the test environment.
  • Review the option pricing model using ML methods developed earlier at the Department. Propose an alternative, efficient option pricing model using ML methods.
  • Design and implement an application which is able to calculate and visualize results systematically: estimated prices over longer time horizon, performance of algorithms.
  • Using the application and the test framework compare the accuracy, performance and sensitivity of theproposal.
  • Discuss the results and document the work.
