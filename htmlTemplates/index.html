This web application is designed for predicting option prices at the expiration date.
To gather data: User needs:
  • Option strike price
  • Current date
  • Option expiration
  • Option ask/bid price
  • Sp&P500 adjusted close
  • Current US risk free rate
  • Vix index
  • (Additionally volatility percentage for Black-Scholes)
These are all freel available information from the internet.


My proposed Models:

model = Sequential()
model.add(Dense(50,activation='elu'))
model.add(Dense(100,activation='elu'))
model.add(Dense(200,activation='elu'))
model.add(Dense(400,activation='elu'))
model.add(Dense(400,activation='elu'))
model.add(Dropout(0.001))
model.add(Dense(200,activation='elu'))
model.add(Dense(100,activation='elu'))
model.add(Dense(50,activation='elu'))
model.add(Dense(1,activation='elu'))
The models are optimized via the Nadam optimizer.
In this web application, an end user is able to calculate both put and call prices.
I've compiled and trained a seperate model based on the options type. When predicted the model automatically selects the appropriate model.


Black-Scholes

For comparing performance, Black-Scholes is used for alternative calculation.
The first universally adopted mathematical model was created by Fisher Black and Myron Scholes in 1973, and was later further developed by Robert Merton. This model, called the Black-Scholes model was initially created to evaluate the European option market put and call options prices [7]. The basic use of the model is to determine the price of an option using log-normal distributions probabilities to calculate volatility
